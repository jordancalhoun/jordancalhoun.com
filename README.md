# Install
1. Clone Repo
2. 
    ```
    pnpm install
    pnpm dev 
    ```

## Template Basics

This template contains the basic setup for 99% of all my SvelteKit projects.  Template includes the following changes to the base `svelete@latest`:

### Modules
- Tailwind CSS
- Sass support
- Typescript
- ESLint & Prettier
- No tests by default

### Pages/etc.
- global `app.scss`
- global `+layout.svelte`
- `Inter var` font
